CSS Filters

CSS filter property are used for graphical effects like drop-shadow, up-shadow, blur, brightness, drop-shadow, invert, saturate, hue-rotate, contrast, etc. Filters are commonly used to adjust the rendering of borders, backgrounds, and images.

CSS filters are very useful and interesting tools used by several professional web developer on their web pages. This powerful visual effects tool can be used to give various special effect like shadowing an image/photo, blurring an image, changing brightness and contrast of an image, etc.

Personally, knowing the usefulness of this CSS filters, I have used these very special tools several times on my webpage for all the images I have upload on it. There are seven images on my webpage which uses the CSS filters: my personal photo (on the very top), and six images of the snakes which I clicked from the Bronx museum.

All the six images have a black shadow, when enlarged, which requires the use of the drop shadow which is a part of the CSS filters. The simple shadowing on the images have brought a great effect to them and have made them more attractive.

Also, as a web developer, I am aware of the fact that the appearance of my webpage, along with the appropriate information, is very important for an interesting and eye-catching webpage. The shadowing effect really draw the attention of the viewer and make the webpage more magnificent.

Image Hover Effects

Another effect that I have widely used on my webpage is the hover effect. As the name suggest, hovering simply mean moving/passing over something. When we use the hover effect on the webpage and hover our computer mouse over the elements on the page, we see a very special effect on that elements like changing the color, size, shape, direction of the image, etc.

Here, on this webpage, I have used hover effect at several places including 6 galley images, the arrow on either side of the images, on the slide changing dots located below the images, and the “send message” tab at the very bottom of the page. To begin with, the first place I have used the hover effect is the 6 images of the sea animals and reptiles, where hovering the mouse over the images would reflect/mirror the whole images.

This effect make the webpage more interactive and interesting for the user. Moreover, I have also added the hover effect on the arrow located on the either side of the six images, where moving the mouse over the arrow would enable the user to change the displayed image.

Also, I have used hover effect on the dot on the bottom of the six images, which can allow the user to change the displayed image. At last, I have used to the hover effect on the very bottom on the “send message” tab, which would allow the user send the written message. All these affects give a new dimension to my webpage and makes it more appealing and attractive.
